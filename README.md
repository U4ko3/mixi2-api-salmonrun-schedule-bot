# サーモンランスケジュール報告bot（for mixi2）

mixi2でサーモンランのバイトスケジュールを報告するためのbotです。

情報は、[Spla3 API]( https://spla3.yuu26.com/)から取得しています。
本botは、当該APIの以下の注意事項に準じます。

- API仕様が変わったり提供を停止する可能性があります。
- このAPIは非公式ですので公式への問い合わせはお控えください。
- システムトラブル等により接続できない場合があります。
- 正しいステージ情報が取得できているはずですが、内容の保証はできません。  
  (スプラトゥーン3 ステージ情報bot と同じデータを使用しています)
- フェス開催時はレギュラーマッチとバンカラマッチのステージ情報が配信されません。フェスマッチのデータをご利用ください。
- 短時間での大量アクセスは行わないでください。

## 実行メモ

botは、８時間ごとに定期的に報告を行います。
また、将来的にbot宛にpostが行われた場合に現在のスケジュール情報を返答します。

定位的な報告は、現在の実行時間と終了時間によりメッセージを変動します。

### 開始間もない、あるいは程度時間が存在する場合（１０時間超過）

現在のスケジュール情報を報告します。

```text
■ 現在のステージ情報
・スケジュール：03/08 09:00　～　03/10 01:00
・ステージ　　：🛣️ すじこジャンクション跡
・オオモノ　　：ヨコヅナ
・ブキ　　　　：
　　わかばシューター
　　パラシェルター
　　モップリン
　　14式竹筒銃・甲
```

### 終了まであと１０時間以内の場合

現在のスケジュール情報を報告します。また、終了時間目安を出力します。

```text
■ 現在のステージ情報　【終了まであと 40 時間！】
・スケジュール：03/08 09:00　～　03/10 01:00
・ステージ　　：🛣️ すじこジャンクション跡
・オオモノ　　：ヨコヅナ
・ブキ　　　　：
　　わかばシューター
　　パラシェルター
　　モップリン
　　14式竹筒銃・甲
```

### もうすぐ終了する場合（５時間以内）

次のスケジュール情報を報告します。

```text
■ 次のステージ情報 開始まであと 40 時間！
・スケジュール：03/10 01:00　～　03/11 17:00
・ステージ　　：🛳️ 難破船ドン・ブラコ
・オオモノ　　：タツ
・ブキ　　　　：
　　スパイガジェット
　　スプラシューター
　　フィンセント
　　リッター4K
```
